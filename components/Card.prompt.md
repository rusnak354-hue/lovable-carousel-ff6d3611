Surface container for grouped content; use for info panels, list rows, and quiet callouts.

```jsx
<Card variant="soft">
  <h4>Безкоштовний аудит</h4>
  <p>Розбір вашої воронки за 45 хвилин.</p>
</Card>
```

Variants: `soft` (milk, 18px radius, long soft shadow — the default), `sharp` (bone, 4px editorial corners), `forest` (dark panel, light text).
