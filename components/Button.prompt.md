Branded action button — forest fill, uppercase, wide letter-spacing; use for primary calls-to-action and form submits.

```jsx
<Button variant="primary" size="md">Записатись на аудит</Button>
<Button variant="outline" pill>Дізнатись більше</Button>
```

Variants: `primary` (forest), `secondary` (sage), `outline` (hairline), `ghost`. Sizes: `sm` / `md` / `lg`. Set `pill` for 999px corners. Press nudges down 1px; hover dims to 0.86 opacity.
