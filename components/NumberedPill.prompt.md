Ordered list / step item — serif italic numeral beside body text in a soft pill. Use for agenda items and numbered takeaways.

```jsx
<NumberedPill number={1}>Закрий ноутбук — система працює без тебе.</NumberedPill>
<NumberedPill number={2} tone="dark">Підніми конверсію на 1% на кожному етапі.</NumberedPill>
```

`tone="light"` (bone) for cream/forest backgrounds; `tone="dark"` (forest) for light backgrounds.
